## 1. 后端开发

### 1.1 Java 开发工程师（初级）

**岗位职责：**
- 负责业务后端服务的设计、开发与维护
- 参与系统架构设计与优化
- 配合前端完成接口联调
- 编写相关技术文档

**任职要求：**
- 1-3 年 Java 开发经验，熟悉 Java 集合、多线程、IO 等基础
- 掌握 Spring Boot、Spring Cloud 框架
- 熟悉 MySQL、Redis 数据库
- 了解 HTTP、TCP/IP 协议
- 具备良好的编码习惯和文档能力

---

### 1.2 Java 开发工程师（高级）

**岗位职责：**
- 负责核心业务系统架构设计与实现
- 主导技术方案评审，解决复杂技术问题
- 指导初中级工程师
- 推动性能优化、代码质量提升

**任职要求：**
- 3-5 年 Java 开发经验，深入理解 JVM 原理
- 精通 Spring Boot、Spring Cloud、Dubbo 等框架
- 具备高并发、大流量系统设计经验
- 熟悉分布式系统原理（CAP、BASE、消息队列、缓存）
- 有源码阅读经验优先（Spring、MyBatis 等）

---

### 1.3 Go 开发工程师

**岗位职责：**
- 负责 Go 后端服务开发
- 参与微服务设计与实现
- API 设计与实现
- 配合测试保障服务质量

**任职要求：**
- 1-3 年 Go 开发经验，熟悉 Go 语法与常用库
- 掌握 Gin、gRPC、ProtoBuf
- 熟悉 MySQL、Redis、MongoDB
- 了解 Docker、Kubernetes
- 有高并发系统经验优先

---

### 1.4 Python 开发工程师

**岗位职责：**
- 负责业务后端服务开发
- 数据处理与接口开发
- 自动化脚本与工具开发

**任职要求：**
- 1-3 年 Python 开发经验
- 熟悉 Django、Flask、FastAPI 框架
- 熟悉 MySQL、Redis
- 了解 Celery、消息队列
- 有数据处理或爬虫经验优先

---

### 1.5 PHP 开发工程师

**岗位职责：**
- 负责 PHP 后端业务开发
- 接口设计与实现
- 数据库设计与优化

**任职要求：**
- 1-3 年 PHP 开发经验
- 熟悉 Laravel、ThinkPHP 框架
- 熟悉 MySQL、Nginx
- 了解 Redis 缓存
- 有高并发项目经验优先

---

### 1.6 Go 开发工程师（基础架构）

**岗位职责：**
- 负责基础架构组件开发（配置中心、服务注册、网关等）
- 参与云原生基础设施建设
- 提升服务稳定性与性能

**任职要求：**
- 3-5 年后端开发经验，精通 Go
- 熟悉 Kubernetes、Docker 原理与实践
- 熟悉微服务架构（服务发现、熔断、限流、链路追踪）

## 4. 算法工程师

### 4.1 推荐算法工程师

**岗位职责：**
- 负责推荐系统算法研发
- 特征工程与模型优化
- 线上效果分析与迭代

**任职要求：**
- 1-3 年推荐算法经验
- 熟悉协同过滤、矩阵分解、深度学习推荐模型
- 掌握 TensorFlow/PyTorch
- 熟悉 Hadoop、Spark 大数据处理
- 有大规模推荐系统经验优先

---

### 4.2 搜索算法工程师

**岗位职责：**
- 负责搜索引擎算法研发
- -query 理解、召回、排序
- 搜索体验优化

**任职要求：**
- 1-3 年搜索/ NLP 经验
- 熟悉 NLP 基础技术（分词、实体识别、语义匹配）
- 掌握向量检索、ANN 算法
- 有 Elasticsearch、Milvus 使用经验优先

---

### 4.3 NLP 算法工程师

**岗位职责：**
- 负责 NLP 相关算法研发
- 文本分类、命名实体识别、情感分析等
- 大模型应用与优化

**任职要求：**
- 1-3 年 NLP 经验
- 熟悉 BERT、GPT 等预训练模型
- 掌握 PyTorch、Transformers
- 有大模型微调、部署经验优先

---

### 4.4 计算机视觉算法工程师

**岗位职责：**
- 负责图像/视频算法研发
- 目标检测、图像分割、OCR 等
- 模型部署与优化

**任职要求：**
- 1-3 年 CV 经验
- 熟悉 CNN、Transformer 等模型
- 掌握 OpenCV、PyTorch
- 有模型部署（TensorRT、ONNX）经验优先

---

### 4.5 机器学习平台工程师

**岗位职责：**
- 负责机器学习平台架构设计与开发

## 📌 使用说明
